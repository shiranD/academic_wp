Personal Profile

Hi, and welcome to my site! I'm currently a PhD candidate at OHSU at the Center for Spoken Language Understanding. My main interests revolve around NLP and assistive technology. You can see some papers I've worked on and repos I've developed. Hope you enjoy!

Research Interests

  • Language Modeling (LM) for users, message decoding for healthy and non-healthy users, Code Switching.
  • Augmentative and Alternative Communication (AAC), Assistive Technology, Brain-Computer Interface (BCI).
  • Continuous-Output prediction, Embedding Space representation.
  • GANs, RL, FSTs.
  • Bioethics and Ethics.
  • Error Analysis, Models Interpretability.
  • Tools for Morphological Languages, Hebrew in particular.
  • Speech Recognition, Speech Synthesis.

Repositories

specializer

2017

A Python library to create Special Fsts as described in OpenFst source code. This specializer library enables sigma, rho, and phi operations over fsts over using the command line operations of OpenFst. To have a small footprint, this functionality was implemented in Cython and included generating C++ files.

Docker Images

2016

Openfst and Pywrapfst library images that provide an openfst with pywrapfst environment on ubuntu 14.4. The most recent images are the pynini_1.4.1 and the fst_1.6.1 and their installation details as well as content is available.

Obfuscation Detection with Deep Learning

2016

The basic code for obfuscation detection provides a trained classifier to analyze and detect the encoding language used to obfuscate parts of code in order to hide there sensitive information or potnetially harmful behavior. (A pdf version of this paper will soon be uploaded to describe the process and evaluation of the method.)

Adapting Textual Techniques to Multimedia Search

2015

The basic code for documents ranking implementing the method described in the paper “OHSU @ MediaEval: Adapting Textual Techniques to Multimedia Search”. This code aims to iteratively score documents given the already chosen documents in order to create a ranked list of documents.

Sequence Classifiers

2015

Code for different sequence classifiers as described in the paper “Textual Prediction of Prosodic Prominence in Spontaneous Speech with Sequence Classifiers”. There are viterbi, greedy first and second order and, local search first second and third order.
